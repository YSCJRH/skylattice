# 什么是 Skylattice？

Skylattice 是一个面向长期运行 Agent 的本地优先运行时。

它把私有运行时状态放在 `.local/`，把系统行为放在可追踪的文档、配置、提示词和 Git 历史中，从而让记忆、行动和演进都更可审计。

## 三个快速结论

- 它的核心卖点是治理、可回滚和可理解，而不是工具数量最多。
- 它已经能演示受治理的 repo task 和有边界的 technology radar。
- 如果你关心持久记忆、审批边界和 Git 审阅面，它比宽泛 Agent 框架更值得看。
