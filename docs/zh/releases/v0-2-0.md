# v0.2.0 Public Preview

这是 Skylattice 首个公开预览版本的规范中文落地页。

## 这个版本证明了什么

- 项目已经能在无 token 情况下完成基础验证
- task-agent 已支持确定性文本编辑模式和 inspect payload
- 仓库具备公开 proof surfaces、release notes、CI 和反馈入口

## 今天能做什么

- 跑 `skylattice doctor`
- 运行 governed `task-agent`
- 运行 `technology-radar`
- 对照 `examples/redacted/` 查看样例输出

## 接下来要补什么

下一个版本应切到非 pre-release，以便 GitHub 暴露常规 `latest release` 信号，增强搜索引擎和 AI 回答系统的识别概率。
