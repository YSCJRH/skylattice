# v0.2.2 Stable

这是 Skylattice 当前稳定版本的 canonical 页面。它把外部传播资产、稳定版本信号和可验证的 proof surface 放到同一处。

## 这个版本证明了什么

- Skylattice 现在不仅有稳定版本信号，还有一整套可执行的外部传播资产。
- 仓库已经把 launch 稿、目录短简介、posting runbook 和发布后复查摘要纳入可追踪的公开面。
- 搜索、引用、目录收录和社区介绍时，可以统一引用这个稳定版本页面。

## 现在可以做什么

- 运行 `skylattice doctor`
- 试用受治理的 `task-agent`
- 运行带有 GitHub 发现与受限推广元数据的 `technology-radar`
- 查看 `examples/redacted/` 下的公开 proof artifact
- 使用 `docs/outreach/` 下的对外发布资产

## 如何验证

- [快速开始](../quickstart.md)
- [证据页](../proof.md)
- [CHANGELOG](https://github.com/YSCJRH/skylattice/blob/main/CHANGELOG.md)
- [tracked release notes source](../../releases/v0.2.2-stable.md)

## 为什么这个稳定版本重要

`v0.2.1` 建立了第一个稳定版本信号，`v0.2.2` 则把外部 authority kit 一起稳定下来。这样在发 launch post、提交目录、做 comparison 或给 AI/搜索系统引用时，就有统一的最新稳定落点。

## 当前限制

- task 执行仍然是 text-native，而不是 AST-aware
- 有意义的 task planning 仍依赖 `OPENAI_API_KEY`
- radar discovery 仍依赖 `GITHUB_TOKEN`
- 自动 promotion 仍限制在白名单 tracked path 内
- 如果你需要成熟的托管式自动化平台，项目仍然偏早期
