# Skylattice

Skylattice 是一个面向构建者的本地优先 AI Agent 运行时，重点不是“更像聊天机器人”，而是让记忆、任务执行和自我演进都能像软件系统一样被检查和审阅。

如果你想找的是一个同时具备持久记忆、受治理的仓库操作能力，以及 Git 原生回滚与审计边界的项目，这个页面就是中文入口。

## 三个快速结论

- Skylattice 不是通用聊天壳，而是一个带运行时状态、账本和治理边界的 Agent 基座。
- 你可以在没有 API Key 的情况下完成基础验证。
- 最快的可信证据来自 quick start、proof 页面和脱敏样例输出。
