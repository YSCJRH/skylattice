# v0.2.2 Stable

This is the canonical public release page for the stable follow-up that packages Skylattice's external authority kit.

## What This Release Proves

- Skylattice now ships a stable authority-building layer that is easy to cite, share, and execute externally.
- The repository tracks launch drafts, directory blurbs, a posting runbook, and a post-release discoverability review alongside the runtime and proof surfaces.
- The stable release signal now reflects both the discoverability baseline and the first outward-facing distribution kit.

## What Works Today

- `skylattice doctor`
- governed `task-agent` runs with deterministic edit modes
- `technology-radar` runs with GitHub discovery and bounded promotion metadata
- Windows-first CI aligned with the tracked validation config
- redacted public proof artifacts under `examples/redacted/`
- tracked launch, outreach, and directory-submission assets under `docs/outreach/`

## Why This Stable Follow-Up Matters

`v0.2.1` established the first stable release signal. `v0.2.2` keeps that signal current while packaging the external authority kit needed for launch posts, directory submissions, and repeatable backlink work.

## Known Limits

- task execution remains text-native rather than AST-aware
- meaningful task planning still depends on `OPENAI_API_KEY`
- radar discovery still depends on `GITHUB_TOKEN`
- automatic promotion remains constrained to whitelisted tracked paths
- the project is still early if you need a broad hosted automation platform
