## What This Preview Was

Skylattice was an early public preview of a local-first agent runtime that kept:

- durable memory in local SQLite-backed state
- operator-facing behavior in tracked docs, prompts, configs, and ADRs
- repo work behind explicit approval and validation boundaries
- bounded self-improvement behind reviewable Git paths and rollback metadata

## What Worked In v0.2.0

- `skylattice doctor` for zero-credential runtime verification
- governed `task-agent` runs with deterministic edit modes and run inspection
- `technology-radar` runs with GitHub discovery, evidence capture, bounded experiments, promotion logs, and rollback metadata
- Windows-first CI using the same tracked validation commands as the runtime
- public-safe sample outputs and walkthroughs under `examples/redacted/`

## Why It Mattered

The preview established proof surfaces, CI, issue templates, and release notes. The later stable cut exists because this preview made the public story auditable first.
