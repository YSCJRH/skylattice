# v0.2.0 Public Preview

This is the canonical public release page for the first Skylattice preview.

## What This Release Proves

- Skylattice is already runnable as a local-first runtime with a no-credential verification path.
- Governed repo tasks now support deterministic text edit modes with inspectable payloads.
- The repository exposes public-safe proof surfaces, release notes, CI, and issue templates for early feedback.

## What Works Today

- `skylattice doctor`
- governed `task-agent` runs with deterministic edit modes
- `technology-radar` runs with GitHub discovery and bounded promotion metadata
- Windows-first CI aligned with the tracked validation config
- redacted public proof artifacts under `examples/redacted/`

## Verify It

- [Quick start](../quickstart.md)
- [Proof](../proof.md)
- [Tracked release notes source](v0.2.0-public-preview.md)

## Known Limits

- task execution remains text-native rather than AST-aware
- meaningful task planning still depends on `OPENAI_API_KEY`
- radar discovery still depends on `GITHUB_TOKEN`
- automatic promotion remains constrained to whitelisted tracked paths

## What Comes Next

The next public version should add a non-pre-release tag so GitHub exposes a conventional `latest release` signal for search engines and AI answer systems.
