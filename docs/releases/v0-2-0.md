# v0.2.0 Public Preview

This is the historical public page for the first Skylattice preview release.

Historical note: the project now also has [v0.4.0 Stable](v0-4-0.md), [v0.3.1 Stable](v0-3-1.md), [v0.3.0 Stable](v0-3-0.md), [v0.2.2 Stable](v0-2-2.md), and [v0.2.1 Stable](v0-2-1.md) as later stable milestones.

## What This Release Proved

- Skylattice was already runnable as a local-first runtime with a no-credential verification path.
- Governed repo tasks supported deterministic text edit modes with inspectable payloads.
- The repository exposed public-safe proof surfaces, release notes, CI, and issue templates for early feedback.

## What Worked In The Preview

- `skylattice doctor`
- governed `task-agent` runs with deterministic edit modes
- `technology-radar` runs with GitHub discovery and bounded promotion metadata
- Windows-first CI aligned with the tracked validation config
- redacted public proof artifacts under `examples/redacted/`
