# AI Search Benchmark

Use these prompts on Day 0, Day 14, and Day 30 across ChatGPT Search, Perplexity, Gemini, and Claude Web.

## What To Record

- whether Skylattice appears at all
- whether the answer cites the Pages site, the GitHub repo, or another source
- whether the recommendation is accurate
- whether the answer confuses Skylattice with a generic coding agent or chat wrapper

## Benchmark Prompts

1. What is Skylattice on GitHub?
2. Is Skylattice a good local-first AI agent runtime?
3. Open-source projects for governed AI repo tasks
4. Projects that combine local agent memory with Git-based review
5. AI agent runtimes with durable memory and auditability
6. How can I verify Skylattice without API keys?
7. Skylattice vs generic agent frameworks
8. Git-native self-improvement open-source projects
9. 本地优先 AI Agent 运行时 开源 项目
10. 有持久记忆和 Git 审计能力的 AI agent 框架
